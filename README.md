## Architecture

- **Scout**: An automated agent that searches for properties using Exa.ai and Tavily.
- **Analyst**: An AI agent that verifies and enriches property data, using Gemini Deep Research for deep dives.
- **Portal**: A web interface for managing the deal flow.

## Directory Structure
- `/backend`: Python FastAPI application (Scout & Analyst agents)
- `/frontend`: Next.js application (User Portal)
